Bem-vindo ao nosso Projeto de HTML com Github!

Esta é um pequena demonstração do uso do Github com HTML.

O que já aprendemos até aqui inclui:

  • Criação de estruturas HTML básicas;
  • Uso de elementos semânticos;
  • Integração com CSS para estilização;
  • Uso de imagens e links;
  • Criação de formulários;
  • Uso de Cards, Layouts e Flexbox;

O que estamos aprendendo agora é:

  • Controle de versão com Git;
  • Colaboração em projetos através do GitHub;
  • Resolução de conflitos de merge;
  • Uso de branches para desenvolvimento paralelo;

O que ainda vamos praticar é:

  • Hospedagem de projetos no GitHub Pages;
  • Uso avançado de Git para gerenciamento de projetos;
  • Integração contínua e deploy automático;
  • Colaboração em equipe usando pull requests e code reviews; e
  • Elaboração do Projeto Integrador por meio do GitHub.
